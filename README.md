# YOLOv5_NCNN
🍅 移动端目标检测，支持iOS、Android，使用NCNN框架。

## iOS:
- Xcode 11.5
- macOS 10.15.4
- iPhone 5s 12.4.5

## android:
- android studio 4.0
- win10 1909
- meizu 16x 8.1.0

安卓请自行确认相关权限是否允许，代码中没有做过多的处理。
