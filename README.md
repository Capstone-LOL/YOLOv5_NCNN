# 移动端目标检测，使用的是YOLOv5的5s模型，摄像头实时捕获视频流进行检测。

## iOS:
- Xcode 11.5
- macOS 10.15.4
- iPhone 5s 12.4.5

## android:
- android studio 4.0
- win10 1909
- meizu 16x 8.1.0

部分代码来自：sunnyden/YOLOV5_NCNN_Android 的开源项目。

如有bug欢迎提issues。安卓请自行确认相关权限是否允许，代码中没有做过多的处理。
